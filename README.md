# Video Game Sales Analysis

## Objective
Analyze global video game sales to identify trends, top platforms, and regional sales patterns.

## Dataset
- Source: Kaggle – Video Game Sales
- Time range: 1980–2016

## Tools Used
- SQL (MySQL): data cleaning and transformation
- Excel: validation and quick analysis
- Python (Pandas, Matplotlib): exploratory analysis
- Tableau: interactive dashboard

## Workflow
1. Imported raw data into MySQL
2. Cleaned and standardized data using SQL
3. Exported clean dataset for further analysis
4. Performed EDA using Excel and Python
5. Built a Tableau dashboard to present insights

## Key Insights
- Video game releases peaked in mid-2000s
- North America dominates global sales
- Platform popularity shifts over time
